정수가 입력 되었을 때, `True`/`False`로 평가해주는 프로그램<br>

`bool()`함수를 이용하면, 입력된 식이나 값을 평가해 불(bool) 형의 값(`True` 혹은 `False`)를 반환해준다. python에서 정수값 `0`은 `False`로 평가되고, 그 외의 값들은 모두 `True`로 평가된다.<br>
