- ###### 3항연산
  3개의 요소로 이루어지는 3항 연산은 `x if C else y`의 형태로 작성이 된다.<br>
  
  `C`: `True` 또는 `False`를 평가할 조건식<br>
  `x`: `C`의 평가 결과가 `True`일 때 사용할 값<br>
  `y`: `C`의 평가 결과가 `True`가 아닐때 사용할 값<br>
<br>

이 3항 연산을 사용하여, 입력된 두 수 중, 더 큰 수를 출력해주는 프로그램을 작성해보면, 다음과 같다.<br>
